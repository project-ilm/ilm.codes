ILM Compatibility & Certification Automation

This document specifies the ILM compatibility and certification model: five levels (ILM Aware, Works with ILM, Built with ILM, ILM Certified, ILM Native), an automated validation pipeline (compilation success, round-trip integrity, canonical identity, protocol conformance, project test suites) delivered as a one-line GitHub Action and a self-certification toolkit, and a public registry. Certification validates protocol conformance, not implementation choices, and self-certification is a declaration of conformance that may require independent audit in regulated environments.
