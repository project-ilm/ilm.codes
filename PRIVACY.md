# Privacy

`ilm.codes` is a static GitHub Pages site. It sets no cookies, runs no analytics or ad trackers, has no
accounts, and stores no visitor data. Hosts (GitHub Pages, and Cloudflare/cdnjs for the 3D library) may keep
standard server logs under their own policies. Issues and pull requests are **public** by nature — do not put
personal or sensitive data in them. Primary sociolinguistic / community data is not accepted.

Rights under India's DPDP Act 2023 and the EU/UK GDPR (access, correction, erasure, grievance) apply to
personal data a controller holds; this site holds none about visitors. Full text: https://ilm.codes/privacy/
